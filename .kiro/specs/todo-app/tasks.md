# 実装計画: TodoリストWebアプリ

## 概要

Flask（Python）をバックエンド、HTML/CSS/JavaScriptをフロントエンドとするシンプルなTodoリスト管理アプリを実装する。タスクデータはJSONファイルに保存し、ブラウザからの操作でタスクの追加・一覧表示・完了状態変更・削除を行う。

## 注意事項

- requirements.md / design.md に記載された内容のみを実装する
- 自動テスト（pytest / Hypothesis / property test）は作成しない
- CI/CD、Docker、デプロイ設定は行わない
- ログイン／ユーザー管理／DB／検索／並び替えなど、要件にない機能は追加しない
- 学校課題として説明しやすいシンプルな構成を維持する
