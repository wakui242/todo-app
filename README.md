# TodoリストWebアプリ

シンプルなTodoリスト管理Webアプリケーションです。タスクの追加・一覧表示・完了状態の変更・削除ができます。

## 機能

- タスクの追加
- タスクの一覧表示（未完了と完了済みを分けて表示）
- タスクの完了状態の変更（未完了⇔完了済み）
- タスクの削除
- データの永続化（JSONファイルに保存）

## 技術構成

- **バックエンド**: Python / Flask
- **フロントエンド**: HTML / CSS / JavaScript
- **データ保存**: JSONファイル（data/todos.json）

## セットアップと起動手順

### 1. 仮想環境の作成と有効化

```bash
# 仮想環境を作成
python3 -m venv venv

# 仮想環境を有効化（Mac/Linux）
source venv/bin/activate

# 仮想環境を有効化（Windows）
venv\Scripts\activate
```

### 2. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 3. アプリの起動

```bash
python app.py
```

### 4. ブラウザでアクセス

ブラウザで以下のURLにアクセスしてください。

```text
http://localhost:5001/
```

## 動作確認手順

アプリが正常に動作することを確認するため、以下の項目を手動でチェックしてください。

### 基本動作

1. **トップページが表示される**
   - ブラウザで `http://localhost:5001/` にアクセスし、ページが正常に表示されることを確認

2. **タスクを追加できる**
   - タスク名を入力して「追加」ボタンをクリック
   - タスクが未完了一覧に表示されることを確認

3. **タスク名が空または空白のみの場合は登録されない**
   - 何も入力せずに「追加」ボタンをクリック
   - エラーメッセージが表示されることを確認
   - 空白（スペース）のみを入力して「追加」ボタンをクリック
   - エラーメッセージが表示されることを確認

### 完了状態の変更

4. **未完了タスクを完了済みに変更できる**
   - 未完了タスクの「完了」ボタンをクリック
   - タスクが完了済み一覧に移動することを確認

5. **完了済みタスクを未完了に戻せる**
   - 完了済みタスクの「未完了に戻す」ボタンをクリック
   - タスクが未完了一覧に移動することを確認

6. **完了済みタスクが見た目で区別される**
   - 完了済みタスクに打ち消し線が表示されることを確認
   - 完了済みタスクがグレー表示されることを確認

### 一覧表示

7. **未完了タスクと完了済みタスクが分けて表示される**
   - 「未完了タスク」セクションと「完了済みタスク」セクションが分かれていることを確認
   - 各タスクが正しいセクションに表示されることを確認

### 削除機能

8. **タスクを削除できる**
   - タスクの「削除」ボタンをクリック
   - 確認ダイアログが表示されることを確認
   - 「OK」をクリックしてタスクが削除されることを確認

### データ永続化

9. **アプリを再起動してもデータが保持される**
   - いくつかタスクを追加
   - アプリを停止（Ctrl+C）
   - 再度 `python app.py` で起動
   - ブラウザをリロードして、タスクが保持されていることを確認

## 停止方法

アプリを停止するには、ターミナルで `Ctrl+C` を押してください。

## 注意事項

- このアプリはローカル環境での使用を想定しています
- ユーザー管理機能はありません（1人での使用を前提）
- データベースは使用せず、JSONファイルにデータを保存します
